Dataset Characteristics¶

Boston house prices dataset in 1978

Number of Instances: 506

Number of Attributes: 13 numeric/categorical predictive. Median Value (attribute 14) is usually the target.

Attribute Information (in order):

  • CRIM: per capita crime rate by town
  • ZN: proportion of residential land zoned for lots over 25,000 sq.ft.
  • INDUS: proportion of non-retail business acres per town
  • CHAS: Charles River dummy variable (= 1 if tract bounds river; 0 otherwise)
  • NOX: nitric oxides concentration (parts per 10 million)
  • RM: average number of rooms per dwelling
  • AGE: proportion of owner-occupied units built prior to 1940
  • DIS: weighted distances to five Boston employment centres
  • RAD: index of accessibility to radial highways
  • TAX: full-value property-tax rate per \$10,000
  • PTRATIO: pupil-teacher ratio by town
  • B: $1000(Bk - 0.63)^2$ where Bk is the proportion of blacks by town
  • LSTAT: % lower status of the population

Linear model: \ $target = \beta_{0} + \beta_{1}crim$

Note: By default, statsmodels fits a line passing through the origin. In other words, it doesn't fit an intercept. Therefore, we need to use add_constant so that it also fits an intercept.

In [9]:
# Performing the regression and fitting the model
model1 = sm.OLS(target, crim1)
fitted_model1 = model1.fit()

  • Linear regression calculates an equation that minimizes the distance between the fitted line and all of the data points.

  • The most common technique to estimate the parameters ($\beta$'s) of the linear model is ordinary least squares (OLS). As the name implies, OLS regression minimizes the sum of the squared residuals.

  • The Formula for the Residual Sum of Squares (RSS): \begin{equation} RSS=\sum_{i=1}^n(y_{i} - f(x_{i}))^2 \end{equation} where

    • $y_{i}$ = the ith value of the variable to be predicted
    • $f(x_{i})$ = predicted value of $y_{i}$
    • n = upper limit of summation
  • residual sum of squares (RSS) = sum of squared residuals (SSR) = sum of squared estimate of errors (SSE)

Interpreting R-squared:

  • R-squared is a statistical measure of how close the data are to the fitted regression line
  • R-squared can take any values between 0 and 1
  • The value shows that 15% of the data fit the regression model
  • The value shows that the regression model accounts for less of the variance
  • Generally, higher r-squared indicates a better fit for the model. However, it is not alwyas the case that a high r-squared is good for the regression model
  • A low r-squared figure is generally a bad sign for predictive models. However, in some cases, a good model may show a small value

Formula for calculating R-squared:\ $$R^2 = \frac{SSR}{SST}$$ where

  • SST = Total sum of squares
  • SSR = Regression sum of squares
  • SSE = Error sum of squares
  • SST = SSE + SSR

Interpreting the coefficients:

  • The crime coefficient in the regression equation is -0.4152
  • For every 1 increase in crime, the target will dicreases with 0.4152
  • If there are 20 crimes, the linear model predicts taht its target is $24.03 - (0.42 * 20) = 15.63$
  • The p-value for the slope is less than 0.05. Hence, we can reject the null hypothesis which allows us to conclude that there is a relationship between crime and target
  • The CRIM predictor variable is statistically significnat since the p-value < 0.05

  • If the model had a high R-Squared, all the points would be close to this diagonal line. The lower the R-Squared, the weaker the Goodness of fit of the model, the more dispersed the points are (away from this diagonal line)

Residuals:

  • Each data has one residual
    • Residual = Observed value Predicted value
  • For linear regression, the residuals always sum to zero when an intercept is included
  • Since the sum of the residuals is zero, the mean of the residuals is zero
  • A residual plot is a graph that shows the residuals on the vertical axis and the independent variable on the horizontal axis. If the points in a residual plot are randomly dispersed around the horizontal axis, a linear regression model is appropriate for the data; otherwise, a nonlinear model is more appropriate
  • We would expect the plot to be random around the value of 0 and not show any pattern
